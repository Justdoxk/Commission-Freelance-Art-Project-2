# BeeplCrab – Portfolio Artistique

## Description

BeeplCrab est un site vitrine one-page développé en freelance pour une étudiante en art. Inspiré de l’univers visuel de Beeple, ce projet propose une expérience immersive à travers une interface moderne, colorée et interactive.

Le site met en avant du contenu biographique et artistique via un système de cards dynamiques et de modals, permettant une navigation fluide sans rechargement de page.

---

## Aperçu

* Interface minimaliste et artistique
* Navigation intuitive
* Expérience immersive
* Mise en avant du contenu visuel et textuel

---

## Fonctionnalités

* Cards interactives (hover + click)
* Modals dynamiques avec contenu personnalisé
* Adaptation automatique des couleurs du modal selon la carte
* Responsive design (desktop / tablette / mobile)
* Chargement rapide (site léger, sans framework)

---

## Technologies utilisées

* HTML5
* CSS3
* Flexbox
* Responsive design
* JavaScript (Vanilla)

---

## Fonctionnement

Le projet repose sur une logique simple et efficace :

Chaque carte contient :

* un titre (<h2>)
* un contenu stocké dans un attribut data-modal-text

Lors d’un clic :

* récupération du contenu via JavaScript
* injection dynamique dans le modal
* récupération de la couleur de la carte
* application au background du modal

Aucun rechargement de page, ce qui permet une expérience fluide.
